# 用户手册

## 1. 运行环境

- Python 3.10 及以上
- Windows 环境下可直接运行；局域网多机演示时，各设备需能互相访问

## 2. 安装依赖

在提交包根目录进入源码目录后执行：

```bash
pip install -r 01_source_code/requirements.txt
```

如果你已经切换到 `01_source_code/` 目录，也可以执行：

```bash
pip install -r requirements.txt
```

## 3. 启动步骤

### 3.1 启动服务端

在项目根目录执行：

```bash
python 01_source_code/server/server.py
```

### 3.2 启动客户端

在另一终端执行：

```bash
python 01_source_code/client/client.py
```

可启动多个客户端窗口，用不同账号登录进行演示。

## 4. 不同电脑之间如何使用

如果服务端和客户端在不同电脑上，需要先把服务端 IP 改成局域网 IP。

修改文件：

- `01_source_code/common/config.py`

把：

```python
SERVER_HOST = '127.0.0.1'
```

改成服务端电脑在当前局域网中的 IP，例如：

```python
SERVER_HOST = '192.168.1.10'
```

要求：

- 所有电脑连接同一个 WiFi 或同一个路由器
- 防火墙未拦截对应端口
- 客户端能访问服务端电脑 IP

## 5. 客户端功能使用说明

### 5.1 注册与登录

1. 打开客户端
2. 输入用户名和密码
3. 新用户先点击“注册”
4. 注册成功后点击“登录”

### 5.2 私聊

1. 在联系人列表中选择目标用户
2. 在输入框中输入消息
3. 点击发送或按回车发送

### 5.3 群聊

1. 切换到群组视图
2. 可创建群组、加入群组、退出群组
3. 进入群聊后即可发送群消息

### 5.4 添加群成员

进入某个群后，可通过群详情区域把其他注册用户加入当前群组。

### 5.5 AI 问答

AI 功能仅在群聊中触发。使用格式：

```text
@AI 你的问题
```

例如：

```text
@AI 介绍一下你自己
```

### 5.6 文件传输

1. 在私聊或群聊中点击发送文件
2. 选择 `.txt` 或 `.md` 文件
3. 接收方会在聊天区看到文件卡片
4. 点击“接收”后开始下载

下载后的文件默认保存在客户端本地下载目录。

## 6. 状态说明

- 在线用户和离线用户会显示不同状态
- 当前未打开的会话收到新消息时，会显示未读提示
- 如果服务器暂时断开，客户端会进入自动重连状态
